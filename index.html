Metroid Prime (GameCube) - GCI Save Patcher

This tool loads a Metroid Prime (GameCube) .gci save file and writes into the data block the 12 bytes that unlock the memory-card-level "extras" (image galleries, ability to start in Hard Mode, Fusion suit, NES Metroid), then recalculates the data block checksum and lets you download the patched file. It should not mess with your game progress.

How to use:

  1. Locate your savefile. If you have Dolphin on Windows, it should be in "C:\Users\yourUsername\AppData\Roaming\Dolphin Emulator\GC\USA\Card A". The folder AppData is likely hidden. Once in your user folder, you can navigate to AppData by typing it in the "url" bar of your file explorer.
  2. Backup your savefile and keep it. I don't want to be responsible for corrupting your data :)
  3. Load the savefile below, apply the patch and replace your original savefile where you found it in the Dolphin directory "Card A". Make sure the folder only contains one .gci file for Metroid Prime, otherwise Dolphin will get confused on which one to load and throw a warning.
  4. Start the game, visit the "Metroid Fusion Connection Bonuses" menu and enjoy!

See https://github.com/robzan8/prime-unlock for details on how the patch works.
